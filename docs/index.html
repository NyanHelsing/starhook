Starhook

Hook into a network beyond the clouds.

Source

Philosophy

The cloud abstracts network resources in a convenient way so that it's accessible from anywhere, but it also centralizes control and ownership of the network, constructing a devide between the consumers of cloud services and the operators of the cloud.

Reclaim the network from the cloud and reimagine it as constellations of interconnected participants.

Starhook creates an entrypoint to out of band network that can operate independently of any cloud provider, enabling consumers to become participants in a network that transcends centralization, and can be extended to whole new regimes of transfer mechanisms.

Install
